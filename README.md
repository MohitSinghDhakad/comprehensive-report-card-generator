# detailedReportCardGenerator
- Generate detailed Report Card for students. 
- Designed to show Total marks, Total percentage, Total rank and Total CGPA.
- Also shows marks, grades and ranks individually in each of the subjects.
- Graph is also plotted showing strengths and weaknesses in each subject.
- For a better idea of performance, studentsâ€™ average and highest marks in each subject are also printed.
- All the data was shown in an organized way using a prettytable library of Python.


# Prerequisite
Following libraries are required to be imported:
- matplotlib library of Python
- PrettyTable library of Python
